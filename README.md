# DevHelp App - Backend

FastAPI backend for DevHelp, a Q&A application.

## Features

- User registration and authentication
- CRUD for questions and answers
- Voting system for questions
- JWT-based authentication
- PostgreSQL database

## Requirements

- Python 3.13+
- FastAPI
- Uvicorn
- SQLAlchemy
- psycopg2-binary
- python-dotenv
- passlib[bcrypt]
- python-jose
